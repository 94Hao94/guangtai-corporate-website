# 官方 Logo 全站替换设计

## 目标

用用户提供的天津光泰科技集团官方 Logo 替换网站现有的自绘品牌图形、组合文字标识和浏览器站点图标，使全站共用入口使用一致且适合各自背景的正式品牌素材。

## 范围与素材映射

| 使用位置       | 当前状态                     | 官方素材               | 呈现方式                                                         |
| -------------- | ---------------------------- | ---------------------- | ---------------------------------------------------------------- |
| 全站固定头部   | 自绘蓝色方形图形加中英文文字 | `光泰科技彩色.png`     | 以完整横版彩色 Logo 替换整个组合标识；保留首页链接和无障碍名称。 |
| 全站固定页脚   | 自绘蓝色方形图形加中英文文字 | `光泰科技白色方形.png` | 以官方白色方版 Logo 替换整个组合标识，确保在深蓝页脚中清晰可见。 |
| 浏览器站点图标 | 自绘 SVG 标识                | 官方灰色方版 Logo      | 用官方方版 PNG 作为 favicon，移除旧 SVG 引用。                   |

素材会复制到 `public/brand/`，由站点以根路径静态引用。保留用户放在项目根目录的原始 logo 文件夹和压缩包，不移动、不删除。

## 组件与样式

`SiteHeader.astro` 与 `SiteFooter.astro` 继续提供网站品牌入口，但将原本由 `.brand-mark`、`b`、`small` 组成的模拟标识改为一张有替代文字的图片。`site-shell.css` 将移除只服务于旧自绘图形和文字的规则，改为控制两种官方 Logo 的合理尺寸、响应式缩放与不裁切显示。

头部的链接名称继续为“天津光泰科技集团首页”，不改变键盘导航或现有浏览器测试语义。页脚品牌图片作为纯品牌展示，替代文字提供公司名称。正文、SEO 标题、结构化数据和版权文本中的公司名称并非 Logo，将保持不变。

## 验证

在构建后的 HTML 中断言头部、页脚和 favicon 都引用官方品牌资源，且不再输出旧 `.brand-mark` 结构。运行格式检查、Astro 类型检查、相关 Vitest 测试和生产构建；若本地预览端口未在监听，则不将浏览器视觉验收伪报为已完成。

## 不在范围内

- 不更改正文、SEO 与结构化数据中的公司名称。
- 不重新设计页面布局、导航或页脚信息架构。
- 不修改用户原始的 `光泰科技logo/` 目录和压缩包。
