# 天津光泰科技集团官网说明文档

## 项目概述

本项目是天津光泰科技集团官网的 Astro 静态站点，承载 AI 应用工厂、解决方案、行业场景、案例与客户、关于光泰和联系入口。当前工作分支为 `feat/mvp-astro-site`，本地开发预览地址为 `http://127.0.0.1:4328`。

## 技术栈

- Astro 7 和 TypeScript 6
- Tailwind CSS 4 与项目内 CSS 样式系统
- Astro Content Collections 管理页面 JSON 内容
- Vitest 4 负责内容与静态输出测试
- Playwright 1.61 负责浏览器端到端测试
- Docker 和非 root Nginx 用于静态部署

## 开发计划

1. 按界面评审优先级逐项处理视觉与布局问题。
2. 每项视觉修改先确认方向，再以内容测试、构建和多视口检查验证。
3. 后续优先处理图片高度稳定性与移动端浏览节奏。

## 问题与解决方案

- 问题：`/cases` 首屏显示带资产名称的内部拼图，损害专业感。
  解决：保留现有版式，只替换 Hero 与 Open Graph 图片，并加入内容合同测试防回归。
- 问题：Astro 开发服务未自动刷新内容集合，修改后仍提供旧图。
  解决：重启当前 worktree 的开发服务并绑定 `127.0.0.1:4328`，确认页面请求新图。
- 问题：迁移前 MVP 输入目录不再存在，遗留合同测试导致全量 Vitest 失败。
  解决：按当前确认的验证范围移除该测试，保留迁移后内容、构建和浏览器测试。

## 部署说明

本地开发使用 `npm run dev -- --host 127.0.0.1 --port 4328`。完整静态验证使用 `npm run verify`；在受限桌面环境中，Playwright Chromium 需要允许正常的本地浏览器进程权限。生产预览使用 Docker Compose 的 8080 端口，构建时必须设置 `SITE_URL`。
